## setup
```bash
# install global package
npm install -g yarn create-react-app

# how to create project react
# example: create-react-app myapp
create-react-app "name project"

# to folder prject
cd "name project"

# start project
npm start or yarn start

# Happy hacking!
# open project
- http://localhost:3000
```
